What does he do?

Tony's Portfolio

Air Bark n Bite

Air Bark n Bite is a clone of Airbnb but for pets. Now when you travel and book a rental you can also book a mini getaway for you pet dog, cat or other pets. This site was built on the React/Redux front-end and a Ruby on Rails backend.

Lyrics Hero

This is a precision timing game, where the player is required to type a letter when it reaches the designated area. The game was inspired by Dance Dance Revolution and the best song in the world, Darude - Sandstrom. Lyrics Hero was made with Vanilla Javascript

Tailor Swift

Tailor Swift uses computer vision to calculate user body proportions and returns a template for shirt tailoring. Tailor Swift is built on a React/Redux front-end, Node.js libraries for image processing and a Ruby on Rails backend.
